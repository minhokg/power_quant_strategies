# power_quant_strategies
Quantitative trading strategies for electricity markets using machine learning, optimization, and statistical modeling.

## Project Overview
- Machine Learning / Deep Learning Pipeline from feature selection to Backtesting
- Model Performance Comparison

## Setup
- install [uv](https://docs.astral.sh/uv/getting-started/installation/)
- run `uv sync` to create the virtual environment and install all dependencies
- run `uv run pre-commit install` to set up git hooks
- if you want to build the package locally (there should be no use case for this), run `uv run python -m build`
- if you want to build the documentation locally (there should be no use case for this), run `cd docs` and then `.\make.bat html`
